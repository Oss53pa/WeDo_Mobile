📱 Comment visualiser l'application?

⚠️ Note importante: React Native n'est pas une application web. Elle nécessite un émulateur mobile ou un appareil physique.
Option 1: Émulateur Android
1. Installez Android Studio
2. Créez un émulateur Android (AVD)
3. Démarrez l'émulateur
4. Dans un nouveau terminal: cd "C:\devs\Wedo-Tontine Digitale\TontineDigital" && npm run android
Option 2: Téléphone Android (Recommandé)
1. Activez le "Mode développeur" sur votre téléphone Android
2. Activez "Débogage USB" dans les options développeur
3. Connectez votre téléphone via USB
4. Exécutez: npm run android
5. L'application s'installera automatiquement sur votre téléphone
Option 3: iPhone (Mac uniquement)
1. Installez Xcode depuis l'App Store
2. Installez CocoaPods: sudo gem install cocoapods
3. Installez les dépendances iOS: cd ios && pod install
4. Exécutez: npm run ios
